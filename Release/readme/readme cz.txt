Snake World Game
================



Jakub Míšek
červen 2008
http://www.sacredware.com/snake-world.aspx



1) Systémové požadavky
......................



- Microsoft Windows 
- 1GHz CPU
- 128MB volné RAM
- DirectX9 kompatibilní 3D grafická karta s podporou pixel shader modelu 2.0 a vertex shader modelu 2.0
- doporučen systém Windows VISTA a graficka karta podporující DirectX 10.0 pro plnou grafickou podporu
  (podpora DirectX 9 je jen okrajová)
- internetové připojení
- zvuková karta

- musí být nainstalován DirectX (March 2008)
použijte instalátor "dxwebsetup.exe", který zařídí stažení a nahrání potřebných souborů do systému
více na http://www.microsoft.com/downloads/details.aspx?displaylang=cs&FamilyID=2da43d38-db71-4c1b-bc6a-9b6652cd92a3



2) Ovládání
...........



- menu - šipky/Enter/Myš/Klávesnice
- hra  - šipky Doleva/Doprava/Nahoru/Dolů
- diskuze během hry - psát text, odeslat Enterem
- rychlé ukončení - Alt+F4



3) Bezpečnost
.............



- aplikace neodesílá žádné osobní údaje
- odesílán je pouze místní kód jazykové sady



4) Varování
...........



- autor není zodpovědný za připadné škody
- použití na vlastní nebezpečí
